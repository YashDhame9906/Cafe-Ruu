# 🍴 Restaurant Web Application

A modern and responsive restaurant website built using HTML, CSS, and JavaScript.  
This project showcases an attractive UI for restaurants including food menus, services, featured dishes, and responsive layouts.

---

## 🚀 Features

- Responsive Website Design
- Modern and Clean UI
- Interactive Navigation Bar
- Food Menu Section
- Featured Dishes Showcase
- About & Contact Sections
- Smooth Scrolling
- Mobile-Friendly Layout

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
